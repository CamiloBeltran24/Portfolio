# 🚀 Christian Beltrán - Personal Portfolio

Bienvenido a mi repositorio de portafolio personal. Este proyecto es mi carta de presentación al mundo, donde muestro mis habilidades como **Frontend Developer**, mi experiencia profesional y los proyectos en los que he trabajado.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5 Semántico:** Estructura accesible y optimizada para SEO.
- **Sass (SCSS):** Arquitectura de estilos modular y escalable.
- **JavaScript (Vanilla):** Lógica interactiva y manejo de formularios.
- **Formspree:** Integración para la recepción de mensajes de contacto.
- **Google Fonts:** Tipografía optimizada (Inter).

## ✨ Características Principales

- **Diseño Responsive:** Optimizado para móviles, tablets y escritorio.
- **UI/UX:** Animaciones fluidas, etiquetas flotantes y modo interactivo.
- **Formulario de Contacto:** Validación en tiempo real y envío asíncrono con `fetch`.
- **Performance:** Carga optimizada de recursos e imágenes.

## 📂 Estructura del Proyecto

```text
/
├── assets/
│   ├── css/      # Estilos compilados
│   ├── IMG/      # Recursos visuales (SVG, Imágenes, Iconos)
│   ├── js/       # Lógica de la aplicación
│   └── sass/     # Código fuente de estilos (Estructura modular)
├── index.html    # Página principal
└── README.md     # Documentación
```

## 🚀 Instalación y Uso Local

1. Clona este repositorio:
   ```bash
   git clone https://github.com/CamiloBeltran24/Portfolio.git
   ```
2. Abre el archivo `index.html` en tu navegador o usa una extensión como **Live Server** en VS Code.
3. Si deseas modificar los estilos, asegúrate de tener instalado Sass para compilar los archivos `.scss`.
